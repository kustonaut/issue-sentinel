    🧩 Features

    // what you get
    🏷️

    Type Classification

    Detects bug, feature-request, question, regression, and docs using 35+ battle-tested regex patterns.

    🎯

    Urgency Scoring

    4-tier priority system (p0–p3) with weighted signals: security vulns, regressions, production outages, CVE references.

    💬

    Sentiment Analysis

    5-class sentiment detection — frustrated, negative, neutral, constructive, positive — to prioritize unhappy users.

    ⚡

    GitHub Action

    Drop kustonaut/issue-sentinel@v1 into any workflow. Auto-labels every new issue. Zero config.

    🤖

    Optional LLM Mode

    Enable --llm for GPT/Claude-powered classification when regex falls short. Graceful fallback to rules.

    🧪

    Fully Tested

    48 unit tests, 100% CI on Python 3.10–3.12, strict mypy, ruff linting. Ship with confidence.
